# FT-003 — Chat Settings Specification

## Overview

Cho phép người dùng cài đặt các thông số cho từng đoạn chat (conversation), bao gồm:
- Tên đoạn chat (name)
- System prompt (systemPrompt)
- Context/contextWindow

Khi gửi tin nhắn lên Gemini, các cài đặt này sẽ được dính kèm theo request.

## User Requirements

1. **Tên conversation**: Người dùng có thể đặt lại tên cho đoạn chat
2. **AI Role / Identity**: Người dùng có thể cài đặt system prompt riêng cho từng conversation
3. **Context / Background**: Người dùng có thể cài đặt context/contextWindow (số token context)
4. **Temperature**: Người dùng có thể cài đặt mức độ sáng tạo (0-1)
5. **Response Length (Max Tokens)**: Người dùng có thể cài đặt độ dài phản hồi tối đa
6. **Yêu cầu**: Người dùng phải đăng nhập (Firebase) mới có thể sử dụng tính năng này

## UI/UX

- Modal hoặc sidebar panel để chỉnh sửa settings
- Input fields: Name, System Prompt (textarea), Context (number/input)
- Save/Cancel buttons

## Database Schema

```prisma
model conversation {
  id            String   @id @default(uuid())
  name          String
  user_id       String?  @map("user_id")
  system_prompt String?  @map("system_prompt")
  context_token Int?     @default(4096) @map("context_token")
  temperature   Float?   @default(0.7) @map("temperature")
  max_tokens    Int?     @default(2048) @map("max_tokens")
  created_at    DateTime @default(now()) @map("created_at")
  updated_at    DateTime @updatedAt @map("updated_at")

  message       message[]

  @@map("conversation")
}
```

Fields:
- `name`: Tên conversation (user có thể đổi)
- `system_prompt`: Custom system prompt cho conversation (nullable)
- `context_token`: Số token context tối đa (default: 4096)
- `temperature`: Mức độ sáng tạo (0-1, default: 0.7)
- `max_tokens`: Số token tối đa cho phản hồi (default: 2048)

## API Design

### PATCH /api/v1/conversations/:id

Cập nhật settings cho conversation.

**Request:**
```json
{
  "name": "Tên mới",
  "systemPrompt": "System prompt...",
  "contextToken": 8192,
  "temperature": 0.8,
  "maxTokens": 4096
}
```

**Response:**
```json
{
  "data": {
    "id": "uuid",
    "name": "Tên mới",
    "systemPrompt": "System prompt...",
    "contextToken": 8192,
    "temperature": 0.8,
    "maxTokens": 4096,
    ...
  },
  "message": "Conversation updated successfully",
  "statusCode": 200
}
```

### GET /api/v1/conversations/:id

Lấy chi tiết conversation (đã bao gồm settings).

## Integration với Chat

Khi gọi API chat:
1. Lấy conversation settings (systemPrompt, contextToken)
2. Gộp vào request gửi Gemini:
   - `systemInstruction`: systemPrompt nếu có
   - `generationConfig`: { contextWindow: contextToken }
